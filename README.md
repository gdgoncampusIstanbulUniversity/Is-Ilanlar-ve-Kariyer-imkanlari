# İş İlanları ve Kariyer İmkanları

Bu repo, **Kariyer & İş İlanları WhatsApp grubumuzda** paylaşılan staj/iş ilanlarını ve kariyer programlarını tek yerde toplamak için tutulur.

> ⚠️ Şartlar ve tarihler değişebilir. Başvurmadan önce mutlaka ilanın kendi sayfasını kontrol edin.

---

## İlan Formatı (standart)

Yeni bir link eklerken aşağıdaki formatı kullanın:

- **[Şirket/Program] – [Pozisyon/Program Adı]**
  - **Hedef sınıf:** (örn. 3. sınıf / 4. sınıf / yüksek lisans / yeni mezun)
  - **Arananlar (kısa):** (örn. İngilizce, haftada X gün, zorunlu staj vb.)
  - **Lokasyon / çalışma modeli:** (İstanbul / hibrit / uzaktan vb.)
  - **Tarih(ler):** (son başvuru / başlangıç / süre; varsa)
  - **Link:** <URL>
  - **Not:** (opsiyonel)
